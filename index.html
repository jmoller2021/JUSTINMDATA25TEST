AlexNet Coral Health Classification System

Differentiating Healthy vs Unhealthy Coral Reefs Using Deep Learning

Author: Justin Moller | Course: Data Science 25

üìä Project Overview

Research Goal

This project implements and compares multiple deep learning architectures (AlexNet, ResNet18, VGG16) to classify coral reef health status. By analyzing various hyperparameters including batch size, learning rate, and data augmentation techniques, we identify the optimal configuration for accurate coral health assessment.

Dataset Information

Training Samples: 1,200 images (600 healthy, 600 unhealthy)

Test Samples: 300 images (150 healthy, 150 unhealthy)

Classes: Healthy Coral, Unhealthy Coral

Image Resolution: 227x227 pixels

üîç Key Findings & Insights

üì¶ Batch Size Impact: Smaller batch sizes (16) showed better performance for coral detail recognition compared to larger batches (64), with a 1.6% accuracy difference. Medium batch size (32) provided the best balance between training stability and performance.
üìä Learning Rate Analysis: A 10x increase in learning rate (0.0001 ‚Üí 0.001) improved accuracy by 1.3%, suggesting the model could benefit from faster convergence. However, care must be taken to avoid overshooting optimal weights.
üé® Data Augmentation Effect: Augmentation techniques (rotation, flipping, color jitter) improved accuracy by 3.9%, demonstrating the model's enhanced ability to generalize to diverse coral appearances and underwater lighting conditions.
üèóÔ∏è Architecture Comparison: ResNet18 outperformed both AlexNet and VGG16 by 6.1% and 2.7% respectively, likely due to its residual connections that enable better gradient flow and feature learning for complex coral textures.
